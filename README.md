# Rabbit Hole Theater v0.31.30-test

## 0.31.30-test complex-interactive-core

- 基于 0.31.29 好看优先版继续调整。
- 不是极限瘦身：保留复杂、精美、交互、媒介本体的正向审美压力。
- 压缩常驻 Prompt：不再注入大段旧版验收、模板名列举、长历史流水账；抽取条目改为紧凑摘要。
- 新增“复杂交互视觉核心”：主视觉结构、前中后景层级、视觉锚点、材质质感、非单调阅读路径、内部可探索入口。
- 加强 JS 真实审查：检测纵向重复骨架、重复内容单元、空间复杂度不足、内部交互入口不足、媒介本体偏弱。
- 代码审查只在内部记录风险；发给模型的纠偏保持抽象，不喂“卡片/问卷/报告/档案”等模板词。
- 不审浅色/白色/米黄本身，只审换汤不换药的骨架复用。
- 代码块急救仍为本地工具，默认关闭。
